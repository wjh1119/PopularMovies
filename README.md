热门新闻（PopularMovies）
===
一个用于浏览最热门或最高分电影的App。
## App设计
- 使用 SyncAdapter 实现定时同步。
- 提供每日推送排名第一电影的功能。
- 支持平板布局。
- 离线时亦可浏览。
## 数据源
数据来源于[The Movie Database](https://www.themoviedb.org/)
